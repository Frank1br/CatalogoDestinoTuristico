# Catálogo de Destinos Turísticos

Aplicativo Android desenvolvido como MVP de um catálogo de destinos turísticos, permitindo cadastrar, listar e explorar destinos via WebView.

## Integrantes

- Amanda Lima
- Frank Oliveira
- Isabela Marchesoni

## Como rodar o projeto

1. Clone o repositório:
   ```
   git clone https://github.com/Frank1br/CatalogoDestinoTuristico
   ```
2. Abra o projeto no **Android Studio**.
3. Aguarde a sincronização do Gradle finalizar.
4. Conecte um dispositivo físico ou inicie um emulador Android (API 24 ou superior).
5. Clique em **Run** (▶) ou use o atalho `Shift + F10`.

## Funcionalidades

- **Tela de Login** — validação com credenciais fixas
- **Cadastro de Destino** — nome, país/região e URL de referência
- **Lista de Destinos** — cards com botões Explorar e Excluir
- **WebView** — abre a URL do destino dentro do próprio app
